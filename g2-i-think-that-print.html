文法まとめ

Slide 1
対象学年：中2　文法事項：I think that の文
形
I think (that) + 主語 + 動詞
意味
「〜だと思います」
よく使う語句
  • I think that
  • I know that
  • that は省略可
  • 文を後ろに続ける
注意点
I think that の文 の形を使う。語順と動詞の形に注意する。
例文
  • I think that he is kind.
  • I think that she likes English.
  • I think that it is interesting.

並び替え＋和文英訳

Slide 4
並び替え
  1. 【 think that / he is kind / I 】私は彼は親切だと思います。
  2. 【 think that / she likes English / I 】私は彼女は英語が好きだと思います。
  3. 【 think that / it is interesting / I 】私はそれは面白いと思います。
  4. 【 think that / Ken is busy / I 】私はケンは忙しいと思います。
和文英訳
  1. 私は彼らはサッカーをすると知っています。ヒント：know / that
  2. 私は彼女は先生だと思います。ヒント：think / that
  3. 私は英語は大切だと思います。ヒント：think / that

解答 Slide3：穴埋め問題

Slide 7
ポイント：I think that の後ろには「主語 + 動詞」の文を続ける。that は省略されることもある。
  1. that後ろに文を続ける。
  2. likeshe が主語なので三単現。
  3. isshe is kind。
  4. will未来の内容。
  5. thinkDo you think ...?
  6. hethat の後ろの主語。
  7. English英語は English。
  8. busy「忙しい」は busy。
