文法まとめ

Slide 1
対象学年：中3　文法事項：関係代名詞 which 物・動物
形
物・動物 + which + 動詞 / 主語 + 動詞
意味
物や動物をくわしく説明する
よく使う語句
  • a book which is popular
  • a dog which has long ears
  • the bike which I use
  • a movie which I watched
注意点
which は物・動物を説明するときに使う。人には who を使う。
○ This is a book which is popular.
× This is a boy which lives here.
例文
  • This is a book which is popular.
  • I have a dog which has long ears.
  • This is the bike which I use every day.

Slide 2
  1. This is a book ___ is popular.
    ア whichイ whoウ whatエ when
  2. I have a dog ___.
    ア who has long earsイ which has long earsウ which have long earsエ which having long ears
  3. This is the bike ___.
    ア who I useイ which use Iウ which I useエ which I using
  4. The cake ___ was good.
    ア who my mother madeイ which my mother makeウ which made my motherエ which my mother made
  5. The song ___ is old.
    ア which I likeイ who I likeウ which like Iエ which I liking
  6. The cat ___ is mine.
    ア who is under the tableイ which is under the tableウ which under the tableエ which are under the table
  7. 正しい表現はどれですか。
    ア the camera who Ken usesイ the camera uses which Kenウ the camera which Ken usesエ the camera which Ken use
  8. 正しい文はどれですか。
    ア The movie who we watched was exciting.イ The movie which watched we was exciting.ウ The movie which we watch was exciting.エ The movie which we watched was exciting.

並び替え＋和文英訳

Slide 4
並び替え
  1. 【 which is / this is / popular / a book 】これは人気のある本です。
  2. 【 which has / I have / long ears / a dog 】私は長い耳を持つ犬を飼っています。
  3. 【 which I use / this is / every day / the bike 】これは私が毎日使う自転車です。
  4. 【 was / the cake / good / which my mother made 】母が作ったケーキはおいしかったです。
和文英訳
  1. 私が好きな歌は古いです。ヒント：song / which
  2. テーブルの下にいる猫は私のです。ヒント：cat / which
  3. 私たちが見た映画はわくわくしました。ヒント：movie / which

Slide 6
ポイント：この単元の範囲内で、形・語順・後続語・否定文・疑問文・意味を確認する。
  1. ア which
    物を説明し、後ろに動詞が続くので which。
  2. イ which has long ears
    動物・物を説明するときは which。
  3. ウ which I use
    目的格でも物を説明する which を使える。
  4. エ which my mother made
    物を説明し、which + 主語 + 動詞。
  5. ア which I like
    song は物なので which。
  6. イ which is under the table
    動物を説明するので which。
  7. ウ the camera which Ken uses
    物 + which + 主語 + 動詞。
  8. エ The movie which we watched was exciting.
    movie は物なので which。
