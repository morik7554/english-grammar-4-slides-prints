# 英語スライドプリント レイアウトサンプル

基準サンプル:

- `english-slide-print-perfect-layout-sample.html`

このレイアウトを、今後の「英語のスライドプリント」作成時の基本形として使う。

## 固定する形式

- HTML印刷用を本命にする
- A4縦1枚を2x2の4分割で印刷
- 4スライド固定
- Slide1: 文法まとめ
- Slide2: 選択問題
- Slide3: 穴埋め問題
- Slide4: 並び替え＋和文英訳
- 教師用解答とレイアウト上の工夫は画面下に置き、印刷時は非表示
- 和文英訳には2語程度のヒントを入れる
- 白黒印刷対応、学校プリント風、書き込み欄優先

## レイアウト方針

- 見出しは大きく、本文は詰め込みすぎない
- 枠線・下線・薄い背景で整理する
- 派手な装飾や色使いは避ける
- イラストより問題量と書き込みやすさを優先する
