文法まとめ

Slide 1
対象学年：中3　文法事項：現在完了（経験用法）
形
have / has + 過去分詞
意味
「〜したことがある」
よく使う語句
  • ever：今までに
  • never：一度も〜ない
  • once / twice：1回 / 2回
  • many times：何度も
注意点
have / has の後ろは「過去形」ではなく「過去分詞」。
× I have went to Kyoto.
○ I have been to Kyoto.
例文
  • I have visited Nara once.
  • She has never seen snow.
  • Have you ever eaten sushi?

並び替え＋和文英訳

Slide 4
並び替え
  1. 【 have / I / Kyoto / been / to 】
  2. 【 has / never / she / snow / seen 】
  3. 【 you / ever / have / sushi / eaten 】
  4. 【 twice / this / read / book / he / has 】
和文英訳
  1. 私は奈良へ一度行ったことがあります。ヒント：been / once
  2. 彼女は一度も野球をしたことがありません。ヒント：never / played
  3. あなたは英語で手紙を書いたことがありますか。ヒント：ever / written
