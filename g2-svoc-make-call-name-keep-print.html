文法まとめ

Slide 1
対象学年：中2　文法事項：SVOC make / call / name / keep
形
動詞 + A + B
意味
「AをBにする・AをBと呼ぶ」
よく使う語句
  • make me happy
  • call him Ken
  • name the dog Pochi
  • keep the room clean
注意点
SVOC make / call / name / keep の形を使う。語順と動詞の形に注意する。
例文
  • The song makes me happy.
  • We call him Ken.
  • They name the dog Pochi.
