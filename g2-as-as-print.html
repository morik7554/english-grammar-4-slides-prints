文法まとめ

Slide 1
対象学年：中2　文法事項：as 〜 as（同等比較）
形
as + 形容詞 / 副詞 + as
意味
「〜と同じくらい…だ」
よく使う語句
  • as tall as
  • as old as
  • as fast as
  • not as ... as
注意点
as 〜 as（同等比較） の形を使う。語順と動詞の形に注意する。
例文
  • Ken is as tall as Yui.
  • This book is as old as that one.
  • He runs as fast as I.

並び替え＋和文英訳

Slide 4
並び替え
  1. 【 is / as tall as Yui / Ken 】ケンはユイと同じくらい背が高いです。
  2. 【 is / as old as that one / This book 】この本はあの本と同じくらい古いです。
  3. 【 runs / as fast as I / He 】彼は私と同じくらい速く走ります。
  4. 【 is / as interesting as math / English 】英語は数学と同じくらい面白いです。
和文英訳
  1. この犬はあの犬ほど大きくありません。ヒント：not as / as
  2. 今日は昨日ほど暑くありません。ヒント：not as / hot
  3. 私のカバンはあなたのと同じくらい新しいです。ヒント：as / new

ポイント：この単元の範囲内で、形・語順・後続語・否定文・疑問文・意味を確認する。
  1. ア asas ... as。
  2. イ asas + 形容詞 + as。
  3. ウ tallas の間は原級。
  4. エ asnot as ... as。
  5. ア fastas fast as。
  6. イ asas new as。
  7. ウ I am as tall as Ken.as tall as。
  8. ア This book is as old as that one.as + 原級 + as。

解答 Slide3：穴埋め問題

Slide 7
ポイント：as ... as の間は原級。否定文では not as ... as を使う。
  1. as / asas ... as の形。
  2. oldas と as の間は原級。
  3. me「私と同じくらい」は as ... as me。
  4. not否定は not as ... as。
  5. yours「あなたのもの」は yours。
  6. cancan + 動詞の原形。
  7. asnot as ... as の2つ目の as。
  8. fastas と as の間は原級。
