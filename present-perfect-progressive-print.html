文法まとめ

Slide 1
対象学年：中3　文法事項：現在完了進行形
形
have / has been + 動詞ing
意味
「ずっと〜し続けている」
よく使う語句
  • for：〜の間
  • since：〜から
  • all morning：午前中ずっと
  • How long：どのくらいの間
注意点
動作が今も続いているときに使う。状態動詞の know / like などは進行形にしにくい。
× I have been knowing him.
○ I have known him for years.
例文
  • I have been studying English for two hours.
  • She has been reading since morning.
  • How long have you been waiting?

並び替え＋和文英訳

Slide 4
並び替え
  1. 【 have / studying / I / been / for two hours 】私は2時間勉強し続けています。
  2. 【 she / reading / has / been / since morning 】彼女は朝から読書をし続けています。
  3. 【 long / waiting / have / how / you / been 】あなたはどのくらい待っていますか。
  4. 【 has / raining / it / been / since last night 】昨夜から雨が降り続いています。
和文英訳
  1. 私は1時間走り続けています。ヒント：been / running（7語）
  2. 彼女は昨日からピアノを練習し続けています。ヒント：practicing / since
  3. あなたはどのくらい英語を勉強し続けていますか。ヒント：How long / studying（8語）

生徒配布用 解答スライド

解答 Slide2：選択問題

Answer 1
ポイント：have / has been + 動詞ing の形を作る。期間は for、始まりは since。
  1. イ been現在完了進行形は have been + 動詞ing。
  2. ウ readingbeen の後ろは動詞ing。
  3. ウ waiting「待ち続けている」は been waiting。
  4. イ haveThey には have を使う。
  5. イ sincelast night は始まった時点。
  6. ア forthree years は期間。
  7. ウ studyingbeen + studying の形。
  8. イ How期間をたずねるときは How long。

解答 Slide3：穴埋め問題

Answer 2
ポイント：主語に合わせて have / has を選び、been + 動詞ing を続ける。
  1. have been studying / forI は have。two hours は期間なので for。
  2. has been reading / sinceShe は has。morning は始まりなので since。
  3. have / been waitingHow long have you been waiting? の形。
  4. have been playing / foran hour は期間。
  5. has been raining sinceIt は has。last night は since。
  6. has been practicing / forKen は has。three years は for。
  7. have been talking sinceafter lunch は始まりの時点。
  8. has been runningall morning は継続時間を表す。

解答 Slide4：並び替え＋和文英訳

Answer 3
ポイント：How long は文頭、疑問文は have / has を主語の前に出す。
  1. I have been studying for two hours.have been + studying の順。
  2. She has been reading since morning.She は has。since morning を文末へ。
  3. How long have you been waiting?How long を文頭に置く。
  4. It has been raining since last night.It has been raining の形。
  5. I have been running for an hour.an hour は期間なので for。
  6. She has been practicing the piano since yesterday.昨日からは since yesterday。
  7. How long have you been studying English?How long + have you been + 動詞ing。

覚えておきたい表現

Plus
日常会話で使いやすい現在完了進行形。勉強・待つ・雨などに使える。
  1. I have been waiting for you.あなたをずっと待っていました。
  2. It has been raining all day.一日中雨が降り続いています。
  3. What have you been doing?ずっと何をしていたの。
  4. I have been thinking about it.そのことをずっと考えています。
  5. She has been working hard.彼女はずっと一生懸命働いています。
  6. How long have you been learning English?どのくらい英語を学んでいますか。
  7. We have been talking for an hour.私たちは1時間話し続けています。
  8. He has been sleeping since nine.彼は9時から眠っています。
