文法まとめ

Slide 1
対象学年：中3　文法事項：受け身の否定文・疑問文
形
be動詞 + not + 過去分詞
Be動詞 + 主語 + 過去分詞 ...?
意味
「〜されません」「〜されますか」
よく使う語句
  • is not called
  • are not used
  • Was ... made?
  • Were ... written?
注意点
受け身の否定文は be動詞の後ろに not。疑問文は be動詞を主語の前に出す。
× This book not is read.
○ This book is not read.
例文
  • This room is not used every day.
  • Is this book read by many students?
  • Were these pictures taken by Ken?

並び替え＋和文英訳

Slide 4
並び替え
  1. 【 used / this room / every day / is not 】この部屋は毎日使われません。
  2. 【 this book / Is / read / by many students 】この本は多くの生徒に読まれていますか。
  3. 【 were not / these pictures / taken / by Ken 】これらの写真はケンによって撮られませんでした。
  4. 【 the windows / Were / cleaned / yesterday 】その窓は昨日掃除されましたか。
和文英訳
  1. この本は英語で書かれていません。ヒント：not / written
  2. このケーキは母によって作られましたか。ヒント：Was / made
  3. これらの部屋は毎日使われますか。ヒント：Are / used

解答 Slide3：穴埋め問題

Slide 7
ポイント：be動詞の位置と過去分詞を確認する。
  1. is not単数 this room なので is not。
  2. are not複数 books なので are not。
  3. was not過去・単数なので was not。
  4. were not過去・複数なので were not。
  5. Is疑問文は Is を前に出す。
  6. Are複数 these cars なので Are。
  7. Was過去・単数なので Was。
  8. Were過去・複数なので Were。
