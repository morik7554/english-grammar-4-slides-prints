文法まとめ

Slide 1
対象学年：中2　文法事項：比較級の変化ルール
形
er / more + 形容詞
意味
比較級の作り方
よく使う語句
  • tall → taller
  • big → bigger
  • easy → easier
  • interesting → more interesting
注意点
比較級の変化ルール の形を使う。語順と動詞の形に注意する。
例文
  • tall becomes taller.
  • big becomes bigger.
  • easy becomes easier.

並び替え＋和文英訳

Slide 4
並び替え
  1. 【 becomes / taller / tall 】tall の比較級は taller です。
  2. 【 becomes / bigger / big 】big の比較級は bigger です。
  3. 【 becomes / easier / easy 】easy の比較級は easier です。
  4. 【 becomes / more interesting / interesting 】interesting の比較級は more interesting です。
和文英訳
  1. hot の比較級は hotter です。ヒント：hot / hotter
  2. new の比較級は newer です。ヒント：new / newer
  3. happy の比較級は happier です。ヒント：happy / happier
