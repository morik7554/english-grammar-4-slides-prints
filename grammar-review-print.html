並び替え＋和文英訳

Slide 4
並び替え
  1. 【 enjoy / I / playing soccer 】私はサッカーをすることを楽しみます。
  2. 【 want / I / to visit Canada 】私はカナダを訪れたいです。
  3. 【 this book / is / easier / than that one 】この本はあの本より簡単です。
  4. 【 I stayed home / because / it was rainy 】雨だったので、私は家にいました。
和文英訳
  1. 私は英語を勉強するために早く起きました。ヒント：to / study
  2. この本はあの本よりおもしろいです。ヒント：more / than
  3. 雨だったので、私たちは家にいました。ヒント：because / stayed

解答 Slide2：選択問題

Answer 1
ポイント：比較・不定詞・動名詞・接続詞 の基本形を選ぶ。
  1. ウ playingenjoy + 動名詞。
  2. イ to visitwant to do。
  3. イ easier比較級。
  4. ア because理由は because。
  5. ウ tallest最上級。
  6. ウ drawing前置詞の後は動名詞。
  7. イ finished現在完了。
  8. ア by受け身の by。

解答 Slide4：並び替え＋和文英訳

Answer 3
ポイント：英語の決まった語順で並べる。
  1. I enjoy playing soccer.enjoy + 動名詞。
  2. I want to visit Canada.want to do。
  3. This book is easier than that one.比較級 + than。
  4. I stayed home because it was rainy.because で理由。
  5. I got up early to study English.目的の to。
  6. This book is more interesting than that one.比較級。
  7. Because it was rainy, we stayed home.because 節。
