文法まとめ

Slide 1
対象学年：中1　文法事項：This is ～ / That is ～
形
This is + 名詞
That is + 名詞
意味
「これは〜です」「あれは〜です」
よく使う語句
  • This is：これは〜です
  • That is：あれは〜です
  • a pen / a book：ペン / 本
  • my bag：私のカバン
注意点
近くのものは this、遠くのものは that。どちらも後ろは is。
× This are a pen.
○ This is a pen.
例文
  • This is a pen.
  • That is my bag.
  • This is an apple.

並び替え＋和文英訳

Slide 4
並び替え
  1. 【 is / a pen / This 】これはペンです。
  2. 【 is / my bag / That 】あれは私のカバンです。
  3. 【 is / this / an apple 】これはリンゴです。
  4. 【 not / that / is / a book 】あれは本ではありません。
和文英訳
  1. これは私の本です。ヒント：This / book
  2. あれはペンではありません。ヒント：not / pen
  3. これはリンゴです。ヒント：This / apple
