対象学年：中2　文法事項：受動態の動詞変化
形
be動詞 + 過去分詞
意味
受け身で使う動詞の形
よく使う語句
  • make → made
  • write → written
  • take → taken
  • speak → spoken
注意点
受動態の動詞変化 の形を使う。語順と動詞の形に注意する。
例文
  • make becomes made.
  • write becomes written.
  • take becomes taken.

並び替え＋和文英訳

Slide 4
並び替え
  1. 【 becomes / made / make 】make の過去分詞は made です。
  2. 【 becomes / written / write 】write の過去分詞は written です。
  3. 【 becomes / taken / take 】take の過去分詞は taken です。
  4. 【 becomes / spoken / speak 】speak の過去分詞は spoken です。
和文英訳
  1. read の過去分詞は read です。ヒント：read / read
  2. clean の過去分詞は cleaned です。ヒント：clean / cleaned
  3. see の過去分詞は seen です。ヒント：see / seen
