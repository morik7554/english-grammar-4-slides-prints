文法まとめ

Slide 1
対象学年：中1　文法事項：動詞のing形の作り方
形
動詞 + ing
意味
現在進行形で使う形
よく使う語句
  • play → playing
  • make → making
  • run → running
  • study → studying
注意点
ふつうは ing。e で終わる語は e を取って ing。短母音+子音字は重ねることがある。
× makeing
○ making
例文
  • play → playing
  • make → making
  • run → running

生徒配布用 解答スライド

覚えておきたい表現

Slide 5
現在進行形でよく使う ing形。
  1. play → playingplay の ing形。
  2. study → studyingstudy の ing形。
  3. make → makingmake は e を取る。
  4. write → writingwrite は e を取る。
  5. run → runningrun は n を重ねる。
  6. swim → swimmingswim は m を重ねる。
  7. sit → sittingsit は t を重ねる。
  8. use → usinguse は e を取る。

Slide 6

解答 Slide3：穴埋め問題

Slide 7
ポイント：動詞の終わり方を見て ing形にする。
  1. playingそのまま ing。
  2. makinge を取る。
  3. runningn を重ねる。
  4. studyingそのまま ing。
  5. swimmingm を重ねる。
  6. writinge を取る。
  7. usinge を取る。
  8. sittingt を重ねる。

解答 Slide4：並び替え＋和文英訳

Slide 8
ポイント：正しい ing形を使って文を作る。
  1. play → playingそのまま ing。
  2. make → makinge を取る。
  3. run → runningn を重ねる。
  4. write → writinge を取る。
  5. I am playing soccer.playing。
  6. He is running.running。
  7. She is writing a letter.writing。
