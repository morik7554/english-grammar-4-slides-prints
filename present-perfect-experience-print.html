文法まとめ

Slide 1
対象学年：中3　文法事項：現在完了（経験用法）
形
have / has + 過去分詞
意味
「〜したことがある」
よく使う語句
  • ever：今までに
  • never：一度も〜ない
  • once / twice：1回 / 2回
  • many times：何度も
注意点
have / has の後ろは過去形ではなく過去分詞。場所に行った経験は been to をよく使う。
× I have went to Kyoto.
○ I have been to Kyoto.
例文
  • I have visited Nara once.
  • She has never seen snow.
  • Have you ever eaten sushi?

並び替え＋和文英訳

Slide 4
並び替え
  1. 【 have / I / Kyoto / been / to 】私は京都へ行ったことがあります。
  2. 【 has / never / she / snow / seen 】彼女は一度も雪を見たことがありません。
  3. 【 you / ever / have / sushi / eaten 】あなたは寿司を食べたことがありますか。
  4. 【 twice / this / read / book / he / has 】彼はこの本を二度読んだことがあります。
和文英訳
  1. 私は沖縄へ一度行ったことがあります。ヒント：been / once
  2. 彼女は一度も野球をしたことがありません。ヒント：never / played
  3. あなたは何回東京を訪れたことがありますか。ヒント：times / ever / visited（8語）

生徒配布用 解答スライド

覚えておきたい表現

Slide 5
日常会話でよく使う現在完了の経験用法。短い形で覚える。
  1. Have you ever been to Tokyo?東京へ行ったことがありますか。旅行の話で使いやすい。
  2. I've never tried it.それを試したことがありません。食べ物や活動に使える。
  3. I've seen it before.前にそれを見たことがあります。映画や写真の話で便利。
  4. Have you ever heard this song?この歌を聞いたことがありますか。
  5. I've been there once.そこへ一度行ったことがあります。
  6. She has met him twice.彼女は彼に二度会ったことがあります。
  7. We've talked about it before.私たちは前にそのことを話したことがあります。
  8. I've never heard that.それは聞いたことがありません。初めて聞いた話に使える。

解答 Slide3：穴埋め問題

Slide 7
ポイント：ever / never / once / many times を手がかりに、have / has と過去分詞をそろえる。
  1. have beenNara へ行った経験なので have been to。
  2. has neverShe は has。「見たことがない」は never seen。
  3. Have / ever疑問文は Have you ... ?、「今までに」は ever。
  4. has writtenKen は has。write の過去分詞は written。
  5. have / manyWe は have。「何度も」は many times。
  6. has seenMy brother は has。see の過去分詞は seen。
  7. have neverThey は have。「一度も〜ない」は never。
  8. Has / beenyour mother は has。場所への経験は been to。

解答 Slide4：並び替え＋和文英訳

Slide 8
ポイント：語順は「主語 + have / has + 過去分詞」。疑問文は Have / Has を前へ出す。
  1. I have been to Kyoto.場所への経験は been to。go の過去分詞 gone はここでは使わない。
  2. She has never seen snow.「一度も〜ない」は has never + 過去分詞。
  3. Have you ever eaten sushi?疑問文は Have you ... ?。ever は have の後ろ。
  4. He has read this book twice.read の過去分詞は read。twice は文末に置ける。
  5. I have been to Okinawa once.「一度」は once。場所への経験は been to。
  6. She has never played baseball.never の後ろは過去分詞 played。
  7. How many times have you ever visited Tokyo?「何回」は How many times。ever を入れて「今までに」の意味を強める。
