文法まとめ

Slide 1
対象学年：中2　文法事項：must の否定文・疑問文
形
must not / Must + 主語 + 原形 ?
意味
「〜してはいけない」「〜しなければなりませんか」
よく使う語句
  • must not
  • mustn't
  • Must I ... ?
  • Yes, you must.
注意点
must の否定文・疑問文 の形を使う。語順と動詞の形に注意する。
例文
  • You must not run here.
  • Must I study?
  • Yes, you must.

並び替え＋和文英訳

Slide 4
並び替え
  1. 【 must not / run here / You 】あなたはここで走ってはいけません。
  2. 【 I / study / Must 】私は勉強しなければなりませんか。
  3. 【 you / must / Yes 】はい、しなければなりません。
  4. 【 must not / swim here / You 】あなたはここで泳いではいけません。
和文英訳
  1. 彼は早く起きなければなりませんか。ヒント：Must / get
  2. 彼らは静かにしなければなりませんか。ヒント：Must / be
  3. あなたはそれを食べてはいけません。ヒント：must not / eat
