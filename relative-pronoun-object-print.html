文法まとめ

Slide 1
対象学年：中3　文法事項：関係代名詞：目的格
形
先行詞 + which / that + 主語 + 動詞
意味
「主語が〜するもの」
よく使う語句
  • that I bought：私が買った
  • which we watched：私たちが見た
  • 目的格の that は省略されることもある
  • 関係代名詞の後ろは主語 + 動詞
注意点
目的格では関係代名詞の後ろに「主語 + 動詞」が来る。
○ the book that I bought
× the book that bought I
例文
  • This is the book that I bought yesterday.
  • The movie which we watched was fun.
  • I like the song that she sings.

  2. 【 the movie / that we watched / was / fun 】私たちが見た映画は楽しかった。

  4. 【 the bag / that he uses / is / new 】彼が使っているカバンは新しいです。
和文英訳
  1. これは私が昨日読んだ本です。ヒント：book / read
  2. 私が使っているペンは新しいです。ヒント：pen / use
  3. 私たちが食べた昼食はおいしかった。ヒント：lunch / ate

覚えておきたい表現

Plus
ものを「私が〜した」と説明するときに使える表現。
  1. This is the book that I bought.これは私が買った本です。
  2. The movie that we watched was fun.私たちが見た映画は楽しかった。
  3. I like the song that she sings.彼女が歌う歌が好きです。
  4. The bag that he uses is new.彼が使うカバンは新しいです。
  5. This is the picture that I took.これは私が撮った写真です。
  6. The cake that we ate was good.私たちが食べたケーキはおいしかった。
  7. The book that Ken likes is long.ケンが好きな本は長いです。
  8. This is the pen that I found.これは私が見つけたペンです。
