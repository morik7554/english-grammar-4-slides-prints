文法まとめ

Slide 1
対象学年：中3　文法事項：受け身
形
be動詞 + 過去分詞
意味
「〜される」「〜された」
よく使う語句
  • by：〜によって
  • is / am / are：現在の受け身
  • was / were：過去の受け身
  • made in：〜製の
注意点
受け身では動詞を過去分詞にする。by の後ろには動作をした人やものを書く。
× This book is readed.
○ This book is read by many people.
例文
  • This book is read by many people.
  • English is spoken in Canada.
  • This room was cleaned by Ken.

並び替え＋和文英訳

Slide 4
並び替え
  1. 【 book / read / this / is / by many people 】この本は多くの人に読まれています。
  2. 【 spoken / English / is / in Canada 】英語はカナダで話されています。
  3. 【 was / this room / cleaned / yesterday 】この部屋は昨日掃除されました。
  4. 【 made / this desk / was / by Ken 】この机はケンによって作られました。
和文英訳
  1. この歌は多くの人に歌われています。ヒント：song / sung
  2. これらの写真は母によって撮られました。ヒント：pictures / taken
  3. このカバンは中国で作られています。ヒント：made / China

生徒配布用 解答スライド

覚えておきたい表現

Slide 5
日常や教科書でよく出る受け身表現。建物・本・言語・製品の説明に使える。
  1. This is made in Japan.これは日本製です。
  2. English is spoken here.ここでは英語が話されています。
  3. The door is closed.ドアは閉められています。
  4. The game was played yesterday.その試合は昨日行われました。
  5. This song is loved by many people.この歌は多くの人に愛されています。
  6. Photos are not allowed here.ここでは写真は禁止されています。
  7. It was written by a student.それは生徒によって書かれました。
  8. The room was cleaned this morning.その部屋は今朝掃除されました。

解答 Slide3：穴埋め問題

Slide 7
ポイント：現在なら is / are、過去なら was / were を使う。動詞は過去分詞。
  1. is readThis book は単数。read の過去分詞は read。
  2. is spokenEnglish は単数扱い。speak の過去分詞は spoken。
  3. were taken複数で過去なので were。take の過去分詞は taken。
  4. was cleaned昨日なので was cleaned。
  5. is sungsing の過去分詞は sung。
  6. was mademake の過去分詞は made。
  7. was brokenbreak の過去分詞は broken。
  8. is mademade in Japan の形。

解答 Slide4：並び替え＋和文英訳

Slide 8
ポイント：受け身の語順は「主語 + be動詞 + 過去分詞 + by ...」。
  1. This book is read by many people.This book is read の語順。
  2. English is spoken in Canada.be動詞 + spoken。
  3. This room was cleaned yesterday.過去なので was cleaned。
  4. This desk was made by Ken.by Ken は文末。
  5. This song is sung by many people.song は単数なので is。
  6. These pictures were taken by my mother.These pictures は複数で過去なので were。
  7. This bag is made in China.「〜製」は made in。
