文法まとめ

Slide 1
対象学年：中3　文法事項：仮定法
形
If + 主語 + 過去形, 主語 + would + 動詞の原形
意味
「もし〜なら、…だろう」
よく使う語句
  • If I were you
  • 語順を確認する
  • 動詞の形に注意する
  • 文全体の意味を考える
注意点
If I were you を中心に確認する。
× If I am you, I will study harder.
○ If I were you, I would study harder.
例文
  • If I were you, I would study harder.
  • If I had time, I would read books.
  • Use the correct word order.

並び替え＋和文英訳

Slide 4
並び替え
  1. 【 if / I / were / you / I would study harder 】もし私があなたなら、もっと勉強するでしょう。
  2. 【 if / I had time / I would / read books 】もし時間があれば、本を読むでしょう。
  3. 【 if / she were here / we would / be happy 】もし彼女がここにいれば、私たちはうれしいでしょう。
  4. 【 I would buy it / if / I had money 】もしお金があれば、それを買うでしょう。
和文英訳
  1. もし私が鳥なら、空を飛ぶでしょう。ヒント：were / would
  2. もし時間があれば、あなたを手伝うでしょう。ヒント：had / help
  3. もし私があなたなら、それを買いません。ヒント：you / not buy

解答 Slide2：選択問題

Answer 1
ポイント：If I were you の基本形を選ぶ。
  1. イ wereIf I were you。
  2. イ wouldwould + 原形。
  3. イ were仮定法では were。
  4. イ had過去形 had。
  5. イ wouldwould play。
  6. イ wouldwould go。
  7. イ wouldwould tell。
  8. イ wouldwould help。

解答 Slide3：穴埋め問題

Slide 7
ポイント：仮定法は If + 過去形, 主語 + would + 原形 を使う。
  1. wereIf I were you。
  2. wouldwould + 原形。
  3. had仮定法では過去形。
  4. couldできたら。
  5. helpwould の後ろは原形。
  6. if条件を表す。
  7. studystudy hard。
  8. gogo there。

解答 Slide4：並び替え＋和文英訳

Answer 3
ポイント：英語の決まった語順で並べる。
  1. If I were you, I would study harder.If I were you。
  2. If I had time, I would read books.would + 原形。
  3. If she were here, we would be happy.If she were。
  4. I would buy it if I had money.if 節は後ろでもよい。
  5. If I were a bird, I would fly in the sky.If I were。
  6. If I had time, I would help you.had + would。
  7. If I were you, I would not buy it.would not + 原形。

覚えておきたい表現

Plus
仮定法 で覚えておきたい表現。
  1. If I were you, I would try it.もし私があなたなら、それを試します。
  2. If I had time, I would read.もし時間があれば、読みます。
  3. If I could fly, I would go there.もし飛べたら、そこへ行きます。
  4. I would help you if I were free.もし暇なら手伝います。
  5. If it were sunny, we would play soccer.もし晴れていればサッカーをします。
  6. If I had money, I would buy it.もしお金があれば買います。
  7. If he knew it, he would tell us.もし彼が知っていれば教えてくれます。
  8. If she were here, we would be happy.もし彼女がここにいればうれしいです。
