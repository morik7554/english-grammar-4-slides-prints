文法まとめ

Slide 1
対象学年：中2　文法事項：助動詞 should
形
should + 動詞の原形
意味
「〜すべきだ」
よく使う語句
  • should study
  • should go
  • should not
  • What should I do?
注意点
助動詞 should の形を使う。語順と動詞の形に注意する。
例文
  • You should study.
  • He should go to bed early.
  • What should I do?

並び替え＋和文英訳

Slide 4
並び替え
  1. 【 should / study / You 】あなたは勉強すべきです。
  2. 【 should / go to bed early / He 】彼は早く寝るべきです。
  3. 【 I / What / do / should 】私は何をすべきですか。
  4. 【 should not / run here / You 】あなたはここで走るべきではありません。
和文英訳
  1. 彼女は医者に会うべきです。ヒント：should / see
  2. 私たちは英語を話すべきです。ヒント：should / speak
  3. ケンは昼食を食べるべきです。ヒント：should / eat

覚えておきたい表現

Slide 5
この文法でよく使う表現。短い文で形を覚える。
  1. You should study.あなたは勉強すべきです。
  2. He should go to bed early.彼は早く寝るべきです。
  3. What should I do?私は何をすべきですか。
  4. You should not run here.あなたはここで走るべきではありません。
  5. She should see a doctor.彼女は医者に会うべきです。
  6. We should speak English.私たちは英語を話すべきです。
  7. Ken should eat lunch.ケンは昼食を食べるべきです。
  8. You should wash your hands.あなたは手を洗うべきです。
